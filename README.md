# Приложение календаря задач с почасовым планированием

Веб-приложение для управления задачами в формате календаря с детализацией по часам.

## Основные функции
- **Почасовой обзор дня** - отображение временных слотов с шагом в 1 час
- **Индикатор текущего времени** - визуальная красная линия, показывающая текущее время
- **Визуализация задач** - задачи располагаются на временной шкале согласно их расписанию
- **Управление задачами** - создание, редактирование, перемещение и удаление задач
- **Динамическое обновление** - автоматическое перемещение индикатора текущего времени

## Технологический стек
- **Next.js**
- **Tailwind CSS**
- **Prisma**
- **API Routes**
